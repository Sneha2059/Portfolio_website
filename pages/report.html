Design, Fabrication and Automation of Industrial 6-DOF Robotic Arm

Final Year Thesis Project | Industrial Automation | On-Demand Manufacturing

Project Type
Robotics & Industrial Automation
Duration
2023-2024
Institution
Tribhuvan University, Institute of Engineering
Technologies
SolidWorks Raspberry Pi Python MATLAB Counterbalance Kinematics

This project developed an industrial-grade 6-degree-of-freedom robotic arm with on-demand manufacturing capabilities for mass/batch production industries. The system integrates mechanical design, automation, and Industry 4.0 concepts to transform manufacturing processes.

Key innovations include a spring counter-balancing mechanism for actuator torque reduction, real-time control via Raspberry Pi, and a complete on-demand manufacturing workflow from client order to automated production.

Project Objectives

  • Streamline the order process with client-side website for real-time order placement and tracking
  • Develop order management software with user and order databases
  • Implement inventory control system for material tracking
  • Design and fabricate 6-DOF robotic arm with integrated spring counter-balancing
  • Simulate robotic arm kinematics using MATLAB environment

Technical Implementation

System Architecture

The project combines mechanical, electrical, and software components:

  • Mechanical: 6-DOF arm with counterbalance mechanism (spring-wire-pulley system)
  • Electrical: Raspberry Pi B+ controlling NEMA stepper motors and MG996R/SG90 servos
  • Software: Python-based control system with Streamlit web interface

Technical Specifications

Component Specification
Actuators NEMA 23 and NEMA 17 Stepper Motors, MG996R and SG90 Servo Motors
Control System Raspberry Pi B+ with RPi.GPIO
Power Supply 5V 2A DC
Software Python with streamlit, sqlalchemy, RPi.GPIO modules
Simulation SolidWorks, MATLAB, ANSYS

Key Innovations

Counterbalance Mechanism

  • Significant reduction in required actuator torque (86.5% for Joint 2, 68% for Joint 3)
  • Enabled use of cost-effective actuators while maintaining payload capacity
  • Integrated spring mechanism within the arm structure

On-Demand Manufacturing System

  • Direct client-to-manufacturer communication system
  • Real-time order tracking and production monitoring
  • Automated inventory management

Kinematic Modeling

  • MATLAB simulation of arm kinematics
  • Denavit-Hartenberg parameters for motion planning
  • Workspace optimization for industrial applications

Project Outcomes

The project successfully demonstrated:

  • Functional 6-DOF robotic arm with counterbalance mechanism
  • Complete on-demand manufacturing workflow
  • Real-time control system using Raspberry Pi
  • MATLAB-verified kinematic models

The system has potential to transform manufacturing by:

  1. Reducing production costs through automation
  2. Enabling direct client-manufacturer communication
  3. Improving production efficiency with real-time monitoring
  4. Providing scalable solution for batch production
