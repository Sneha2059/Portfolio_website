3D CAD Model Classification using Graph Neural Networks

Research project at Autolab Technologies

Project Type
Research Internship
Duration
June 2024 - Present
Organization
Autolab Technologies
Technologies
GNNs Deepwalk Python PyTorch

This ongoing research project at Autolab Technologies explores the application of Graph Neural Networks (GNNs) for classifying 3D CAD models. Traditional methods for CAD model classification often rely on manual feature extraction or convolutional neural networks (CNNs) that process voxelized representations, which can be computationally expensive and lose important structural information.

Our approach treats CAD models as graphs, where nodes represent geometric primitives (like faces, edges, or vertices) and edges represent their relationships. This graph representation preserves the rich topological and geometric information inherent in CAD designs while being more efficient to process than voxel-based methods.

Potential Applications

This research has several practical applications in engineering and manufacturing:

  • Automated organization of CAD model libraries
  • Quality control by detecting design anomalies
  • Recommendation systems for similar parts
  • Accelerated design retrieval in PLM systems
  • Support for generative design workflows

Key Features

  • Graph-based representation preserves design intent
  • Scale-invariant classification
  • Handles both parametric and mesh-based CAD
  • Interpretable attention weights show important features

Challenges

  • Handling large assemblies
  • Dealing with imperfect CAD geometry
  • Limited labeled datasets
  • Computational efficiency for real-time use

Future Work

  • Extend to generative tasks
  • Incorporate physical simulation data
  • Develop multi-modal approaches
  • Create benchmark datasets
