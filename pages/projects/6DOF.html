Design & Automation of 6-DOF Robotic Arm with On-Demand Manufacturing

Final Year Engineering Project | Team Project | Industrial Automation

Project Type
Robotics & Industrial Automation
Duration
2023-2024
Institution
Tribhuvan University, Institute of Engineering
Technologies
SolidWorks Raspberry Pi Python MATLAB Counterbalance Mechanism Kinematics

This project designed and fabricated a cost-effective 6-degree-of-freedom (DOF) robotic arm with a spring counterbalance mechanism (CBM) and integrated on-demand manufacturing capabilities. The system combines mechanical design, automation, and web-based control to revolutionize small-scale industrial production.

Key innovations include an 86.5% reduction in actuator torque requirements through the CBM, real-time control via Raspberry Pi, and a client-facing order management system. The robotic arm can handle payloads up to 2kg with precision movement controlled through inverse kinematics.

Methodology

Our development process followed these stages:

  1. Mechanical Design: 3D CAD modeling in SolidWorks with iterative prototyping for weight reduction and CBM integration.
  2. Counterbalance Mechanism: Implemented spring-wire-pulley system with double parallelogram linkage to maintain reference planes.
  3. Electronics: Raspberry Pi B+ with parallel threading for synchronized control of 6 stepper/servo motors.
  4. Kinematics: Developed forward/inverse kinematic models using Denavit-Hartenberg parameters, validated in MATLAB.
  5. Software Integration: Python-based web interface (Streamlit) for order management and real-time arm control.

Innovations

Counterbalance Mechanism

  • Spring stiffness calculations: 847 N/m (Joint 2), 320 N/m (Joint 3)
  • Double parallelogram mechanism maintains independent reference planes
  • Actuators mounted at base to reduce rotational inertia

On-Demand Manufacturing System

  • Streamlit web interface for order placement/tracking
  • SQLite database integration
  • Automated path planning via inverse kinematics

Kinematic Model

  • Closed-form inverse kinematic solution for RPPPYR configuration
  • MATLAB validation with Peter Corke's Robotics Toolbox
  • Workspace analysis via 10,000-point Monte Carlo simulation

Conclusions & Future Work

The project successfully demonstrated:

  • Cost reduction through innovative counterbalancing (86.5% torque savings)
  • Precision control via kinematic modeling and Raspberry Pi automation
  • Industrial 4.0 integration with on-demand manufacturing workflow

Recommended improvements:

  1. Implement belt tensioners to prevent slippage
  2. Upgrade to aluminum frame for increased rigidity
  3. Add force feedback sensors for closed-loop control
  4. Expand order management system with AI-driven scheduling
